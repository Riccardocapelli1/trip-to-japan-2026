# Spec: Daily Planning & Discovery

## Goal
Enable the traveler to drill down into a specific day to find actionable suggestions for sightseeing, dining, and shopping.

## User Persona
- **The Explorer**: Looking for the best spots in the current city.
- **The Romantic**: Looking for personalized notes and special touches.

## Success Criteria
- [ ] Clicking/Viewing a specific date card reveals "Must See" spots.
- [ ] "Local Spots" (Restaurants and Stores) are displayed for each day.
- [ ] Personalized notes (Ricky's Notes) are highlighted to add emotional value.

## Detailed Flow
1. **Selection**: User navigates to the card corresponding to the current date.
2. **Review**: User reads the "Must See" list (e.g., Ghibli Museum).
3. **Dining**: User checks "Local Spots" for highly-rated restaurant suggestions.
4. **Personalization**: User reads the specific note for the day (e.g., "bacino") to feel the personal connection to the plan.
