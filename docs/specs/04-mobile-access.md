# Spec: Mobile On-the-Go Access

## Goal
Ensure the itinerary is easily accessible and readable on mobile devices while the travelers are navigating the streets of Japan.

## User Persona
- **The Navigator**: Checking the next spot while walking or on a train.
- **The Decider**: Quickly looking for a place to eat nearby.

## Success Criteria
- [ ] Layout collapses gracefully into a single-column view.
- [ ] High contrast text is readable in outdoor lighting.
- [ ] Interactive elements (buttons, search) are touch-friendly.

## Detailed Flow
1. **Access**: User opens the website URL on their smartphone.
2. **Navigation**: User uses touch gestures to scroll through the vertical timeline.
3. **Check**: User quickly identifies the "Base City" and "Intensity" for the day.
4. **Action**: User finds a restaurant name in the "Local Spots" section to input into a map application.
